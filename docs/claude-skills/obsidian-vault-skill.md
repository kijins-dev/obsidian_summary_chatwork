## スキル概要

NaokiのObsidian vault（C:\Users\kijin\Desktop\dev\Obsidian）を操作する際のルールとベストプラクティス。

## フォルダ構造

```
Obsidian/
├── 00_タスクBot/          # n8n自動管理（手動編集禁止）
├── 00_受信箱/             # 一時保存・未整理
├── 01_デイリーノート/      # 日次記録（YYYY-MM-DD.md）
├── 02_プロジェクト/        # プロジェクト別記録
├── 05_会議記録/           # Tactiq自動生成
├── 07_テンプレート/        # テンプレートファイル
├── 08_Chatworkログ/       # n8n自動生成（日次）
└── 09_リアル会議記録/      # PLAUD自動生成
```

## 命名規則

| 種類 | 形式 |
|------|------|
| デイリーノート | `YYYY-MM-DD.md` |
| 会議記録 | `YYYY年MM月DD日_タイトル_summary.md` |
| Chatworkログ | `YYYY-MM-DD.md` |
| 開発記録 | `YYYY-MM-DD_内容.md` |

## デイリーノート更新ルール

1. **既存内容を削除しない** - 追記のみ
2. **セクションがなければ作成**
3. **既存セクションには末尾追記**

## タスク記法

```markdown
# 基本形式（n8n互換）
- [ ] タスク内容（M月D日 HH:MM）

# Tasksプラグイン拡張
- [ ] タスク 📅 2026-01-20 ⏫
```

## エンコーディング

- UTF-8（BOMなし）
- LF改行

## 禁止事項

1. 00_タスクBot配下の手動編集
2. 自動生成ファイルの構造変更
3. BOM付きUTF-8の使用
